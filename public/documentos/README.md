## Formatos Aceptados
- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- PowerPoint (.ppt, .pptx)

## Instrucciones
1. Coloca los archivos en la carpeta correspondiente a su área
2. Usa nombres de archivo descriptivos (ej: `informe-anual-2025.pdf`)
3. Los documentos aparecerán automáticamente en la página web
